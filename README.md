## О проекте
Проект представляет собой **систему учета сотрудников и их должностей в организации**, а также систему выдачи выговоров, премий и заданий сотрудникам.

### Используемая архитектура
В качестве архитектуры используется язык программирования Python, веб-фреймворк Flask, система API Postman, система база данных SQLite.

### Цели проекта
Основными целями проекта являются *централизация* и безопасное *хранение кадровых данных*, *автоматизация процессов* обновления информации, *обеспечение контролируемого доступа* к конфиденциальным данным в зависимости от должностных обязанностей, а также упрощение *формирования отчетности* для руководства.

### Пользователи в системе
Пользователями выступают работники организации с различными уровнями доступа:
1. Роль *Сотрудник* позволяет только просматривать свой профиль, включая свои задачи и выговоры.
2. Роль *Глава отдела* предоставляет права на просмотр профилей всех сотрудников в своем отделе, на выдачу задач сотрудникам, на выдачу выговоров и премий.
3. Роль *HR* имеет полный доступ на чтение и запись ко всем данным сотрудников, включая регистрацию новых сотрудников.

### Основыне функции системы
- авторизацию пользователей по ролям
- многоуровневое отображение данных в зависимости от прав доступа
- редактирование профилей сотрудников
- ведение истории изменений критичных данных.
